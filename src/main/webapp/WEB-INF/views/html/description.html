O aplikacji

Wykorzystane technologie:

Podstawowe:

Java 8, Spring MVC 5.0.6, Hibernate 5.2.17, Apache Maven 3.3.9, Apache Tomcat 8.0.52, Thymeleaf 3.0.9

Webowe:

HTML5, CSS3, Javascript, jQuery UI + DataTables, Bootstrap

Bazy danych:

Aplikacja powstała przy pomocy bazy danych PostgreSQL 10.3

(testowana także na Oracle12c oraz MySQL 5.7)

Ponadto :

Kodowanie miało miejsce pod kontrolą systemu Manjaro Linux 17.0.6 oraz środowiska programistycznego Intellij IDEA Ultimate2018

Stronę wizualną projektu przygotowano za pomocą programu Webstorm 2018 Działanie aplikacji sprawdzono także pod kontrolą systemu Windows 10 z zainstalowanym środowiskiem programistycznym Eclipse.

Testy nie pokazały niewłaściwego działania aplikacji.

Dokumentacja została stworzona za pomocą programu Javadoc.
